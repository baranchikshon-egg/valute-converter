**Как получить API-ключ:**
1. Зарегистрируйся на https://www.exchangerate-api.com/
2. Подтверди email — ключ придёт на почту.
3. Вставь ключ в файл currency_converter.py в строку API_KEY = "твой_ключ"

**Как запустить:**
1. Установи Python с сайта python.org
2. Открой командную строку и напиши: pip install requests
3. Запусти файл currency_converter.py двойным щелчком

**Пример использования:**
Выбери USD → EUR, введи 100, нажми «Конвертировать». Увидишь результат и запись в истории.
